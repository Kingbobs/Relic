3) Its easy To Config For exaple!

**- [Common]:
  - Name of relic when receiving the relic
  - Name: "&cCommon Relic"
  - Item Lore
  - Lore: "&cPlace the relic to claim! Ok!"
  - Messages to run
  - Message:
    - Receive: "You found a Common relic"
    - Opening: "Opening Common relic"
